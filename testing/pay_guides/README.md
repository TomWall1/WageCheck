# Pay Guide PDFs

Place the official FWO pay guide PDF for each award here, named by award code:

```
MA000100.pdf   (SCHADS)
MA000009.pdf   (Hospitality - HIGA)
MA000003.pdf   (Fast Food)
MA000004.pdf   (Retail)
...
```

Source: https://www.fairwork.gov.au/pay-and-wages/paying-under-an-award/pay-guides

These PDFs are the source of truth for Stage 0 verification. The exhaustive
test agent reads each PDF, extracts every rate/allowance/condition, and verifies
that the database and calculator cover every number in the guide.
